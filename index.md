# Erasmus Cybersecurity Lab

Welcome to my cybersecurity lab documentation.

This website contains my personal notes, experiments, and hands-on practice as I learn and explore different areas of cybersecurity and ethical hacking.

---

## About This Lab

In this lab, I practice cybersecurity concepts using virtual machines and security tools. The goal is to understand how systems work, how vulnerabilities are discovered, and how they can be secured.

All experiments documented here are performed in **controlled lab environments** for educational purposes.

---

## Topics Covered

### Web Security
- SQL Injection
- Cross-Site Scripting (XSS)
- File Inclusion (LFI/RFI)
- Authentication Bypass

### Network Security
- Network Scanning with Nmap
- Packet Analysis with Wireshark
- Network Enumeration

### Linux Security
- Linux File System
- File Permissions
- Privilege Escalation

### CTF Writeups
- Practice machines
- Capture The Flag challenges
- Vulnerability walkthroughs
